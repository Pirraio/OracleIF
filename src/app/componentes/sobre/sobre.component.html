Sobre o Projeto

Projeto desenvolvido durante a disciplina de Programação para Internet em conjunto com disciplina de Projeto de Desenvolvimento de Software durante o 4º ano do Curso Técnico em Informática. Esse projeto utiliza em seu frontend o framework Angular, no backend é utilizado nodejs e express, um framework nodejs para a construção de servidores web. Os componentes do grupo são: Anthony Samuel, Arthur Emanuel, Damião Victor, Gabriel Teixeira e Ian Barbosa. Os professores orientadores são Bruno Gurgel e Álvaro Hermano, respectivamente em PPI e PDS.
